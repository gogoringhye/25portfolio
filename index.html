소비자에서 창작자로의 전환

  • 즐기는 사람에서
  • 관찰하고 배우는 사람으로
  • 창작하는 사람으로

  • 즐기는 것에서 시작됐습니다.

    OTT 서비스, 스포츠 경기, 공연, 음악 등 다양한 콘텐츠를 즐기며 자라온 저는 언제나 감상에 몰입하는 데 익숙한 소비자였습니다. 창작물이 주는 즐거움과 감동은 제 일상 속 소중한 쉼표가 되어 주었죠. 그러던 어느 날, 문득 이런 생각이 들었습니다. '나도 누군가에게 이런 즐거움과 감동을 줄 수 있는 방법은 없을까?' 이 질문을 시작으로, 단순히 감상하는 사람에서 벗어나 창작자가 되어, 사용자가 경험하고 감동을 느낄 수 있는 공간을 디자인하고, 그 안에 의미를 담아내고 싶다는 욕심이 생겼습니다.

    감상의 시작이 궁금하다면?

  • 관찰하고 배우는 시간이었습니다.

    좋아하던 콘텐츠를 단순히 즐기기보다, 그 안에 담긴 이야기의 구조나 감정을 끌어내는 연출 방식 등을 관찰하게 됐습니다. 이러한 시선은 제가 만든 화면이나 콘텐츠에도 자연스럽게 녹아들었고, 사용자 중심의 웹 화면을 구성하는 데 중요한 밑거름이 되었습니다.

    내가 배운 콘텐츠의 힘은?

  • 창작의 도구를 만났습니다.

    무엇으로 나만의 이야기를 풀어낼 수 있을까 고민하던 중, 퍼블리싱과 프론트엔드 개발을 알게 되었습니다. 코드를 통해 의미 있는 경험을 시각적으로 표현하고, 화면에 생명을 불어넣는 그 과정에서 깊은 몰입을 느꼈습니다. 소비자였던 내가, 이제는 누군가에게 영감을 줄 수 있는 창작자가 되고자 합니다.

    퍼블리싱이 내게 준 변화는?
# React # Node.js # JSX # SPA

React를 이용하여 실제 페이지는 하나지만, 사용자에게는 여러 페이지가 있는 것처럼 보이도록하여 페이지 전체를 새로고침하지 않고, 필요한 부분만 바꿔서 앱처럼 빠르게 동작하는 Single Page Application 입니다.

Web Design That Preserves Values Amidst Change

빠르게 변화하는 기술 속에서도, 사용자의 경험과 웹의 본질을 존중하는 마음을 지키고자 합니다.

  • 설계에 앞서 고민부터

    왜 그렇게 만드는지에 대한 고민을 바탕으로, 프로젝트에 적합한 웹 구조와 디자인을 설계합니다.

  • 기본을 지키는 자세

    기본 원칙을 준수하고, 표준과 접근성을 고려합니다.

  • 지속적인 성장

    사용자가 진정으로 필요로 하는 가치를 제공하기 위해 끊임없이 발전하고, 변화에 유연하게 대응할 수 있는 능력을 기르고 있습니다.
